## 📌 Overview

This project was developed in collaboration with **Husqvarna AB** to optimize the mechanical properties of austempered ductile iron (ADI) components using machine learning — replacing costly and time-consuming physical experiments with a data-driven, interactive platform.

The result is a **Streamlit web application** that allows engineers to explore optimal austempering parameters (austenitizing temperature, austempering temperature, and time) and instantly predict mechanical outcomes such as Ultimate Tensile Strength (UTS) and Yield Strength (YS).

---

## 🎯 Problem Statement

Traditional austempering process design relies on iterative physical trials — an expensive, slow, and resource-intensive approach. This project replaces that process with a machine learning pipeline that:

- Predicts mechanical properties from process parameters
- Generates and filters optimal parameter combinations
- Provides explainable AI-driven analysis of results

## 🔬 Methodology

### 1. Data Collection & Preprocessing
Experimental data from physical austempering trials was collected, cleaned, and engineered into features suitable for supervised learning.

### 2. Predictive Modeling
Multiple regression models were trained and evaluated to predict UTS and YS from:
- Austenitizing temperature
- Austempering temperature
- Austempering time

### 3. Multi-Objective Optimization
A parameter space search (scaled 0–100) generates candidate solutions. Users then filter these using an **interactive parallel coordinates plot** to identify the Pareto-optimal configurations matching their engineering constraints.

### 4. AI Analyst
An integrated **Azure OpenAI GPT-4** module (via Semantic Kernel) generates automated, human-readable reports explaining why specific parameter combinations yield superior mechanical properties.
